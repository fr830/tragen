** README file of Tragen (OPC-UA traffic generator) **

# Building:

**Installation:**
```
$ make install
```
or, (to generate docs as well)
```
$ make
```

**Uninstallation:**
```
$ make uninstall
```
To remove also Sphinx - the documentation generator:
```
$ make clean_all
```

**Generate documentation:**
```
$ make doc
```
